## Usage

```bash
❯ curl -X GET https://xcdaas.now.sh/api/discover/:domain
```

## Development

### Prerequisites

- [Node.js](https://nodejs.org/en) `v10.15.3`
- [Yarn](https://yarnpkg.com/en/)

### Running

- `❯ yarn`
- `❯ yarn dev`
